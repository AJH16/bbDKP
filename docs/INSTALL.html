Description

bbDKP is a Dragon Kill system Mod for your phpBB3 Bulletin board, developed from EQDKP.

It is a multi-pool loot distribution system that supports Standard DKP, Time based DKP, Zero sum and EPGP.

It integrates Game, Guild and DKP management functions into your phpBB3 Forum. Your users gain single sign on to dkp and phpBB3, and you as an administrator or raidleader get a clean interface, robust admin facilities and less integration headaches.

Feature list : Portal with widgets, Dashboard, Statistics, Events, Raids, Member, Items, Roster, Bossprogress page, ACP and UCP modules. multiple games : World of Warcraft, The Lord of the Rings Online, Dark Age of Camelot, Vanguard - Saga of Heroes, EverQuest I/II, Warhammer Online, Final Fantasy XI, AION.

There are a number of plugins, like Raidtracker, bbTips, Memberimport, Apply, Raidplanner (under development).

Further development requests, support questions are welcome at http://www.bbdkp.com. As of the current release, it does not (yet) make coffee ;)

Languages included: Englisch (en), French (fr), Deutsch (de)

Available Plugins :
  • Armory Import 1.0.3 (Wow & Aion)
  • bbTips (Wow) 0.4.0
  • Raidtracker 0.3.0
  • Recruitment 1.2.7
  • Raidplanner 0.1.0

Requirements

  1. phpbb3.0.5 or higher
  2. PHP 5.2 -> 5.3.x / Libxml 2.6.0 (simplexml usage)
  3. Curl (preferred) or url_fopen or fsockopen should be available on your server.
  4. ftp and founder admin access on your phpbb installation.

Installation

Adding bbDKP to phpBB3 is easy. you need to do the edits in your core phpbb files, copy bDKP and your modded files to your server, and then run the database installer.

Step-by-step Guide

New installation using Automod

  1. Unzip the bbdkp zip file
  2. Upload it to your Site, into /store/mods/
  3. Launch automod, click in "bbDKP", and click the "Install" link. this will copy the files, and perform the necessary edits.
  4. You need to copy the content of root/images folder to your Images folder manually : this is not done by Automod because it can't handle it due to the folder size (it's about 25 mb)
    from="root/images/bossprogress/*.*" to="images/bossprogress/*.*"
    from="root/images/class_images/*.*" to="images/class_images/*.*"
    from="root/images/progressbar/*.*" to="images/progressbar/*.*"
    from="root/images/race_images/*.*" to="images/race_images/*.*"
    from="root/images/recruitblock_images/*.*" to="images/recruitblock_images/*.*"
    from="root/images/roster_classes/*.*" to="images/roster_classes/*.*"
    from="root/images/roster_portraits/*.*" to="images/roster_portraits/*.*"
    from="root/images/tooltip/*.*" to="images/tooltip/*.*"
  5. Then surf to /install/installdkp.php, and you will see the umil installation page. This will install the acp modules, database tables, and clear the caches (template, theme, imagesets)

Upgrading to 1.2 from 1.1.2.2

  • Before upgrading bbDKP, you need to uninstall all plugins.
  • If you want to upgrade manually, you just need to follow the instructions in contrib/update_1122_to_12.xml :
    • first you copy all files to your webserver, replacing any old files. (Tip : it is better to just delete and replace)
    • then you apply the (few) changes to the core phpbb files
    • then you run the DB updater (/install/installdkp.php)
  • Using Automod
    • (Recommended) Uninstall bbDKP by clicking "Uninstall" in Automod, then follow the instructions for a New install.
    • If you want to *update* Using Automod, we prepared a separate download with a changed files MODX package. Refer to the instructions in that download.

Upgrading from 1.0.8-1.1 to 1.2

  1. Completely remove all old files and all old bbDKP edits from your phpbb installation.
  2. Follow the Clean install instructions

Quick Deactivation

  1. should you wish to Deactivate this Mod quickly you can open config.php
    2. change @define('EMED_BBDKP', true); to //@define('EMED_BBDKP', true);
