Description

bbDKP for phpBB3 brings together the DKP management facilites of EQdkp and integrates them fully with phpBB3's facilities. This means your users gain single sign on to dkp, and you gain a clean interface, robust admin facilities and less headaches.
Games supported : Aion, Daoc, Eq, Eq2, ffxi, lotro, vanguard, warhammer, Wow.
Features : Multiple Dkp Pools, Multiple Guilds, Standings, Raids, Events, Item values, Item history, Raid Stats, Bossprogress, Roster, backup, import from Eqdkp.
Available Plugins :
  • Armory Import (Wow & Aion)
  • bbTips (Wow)
  • Ct_Raidtracker (wow)
  • Recruitment (Wow)

Requirements

  1. phpbb3.0.5 or higher
  2. PHP 5.1.0 and Libxml 2.6.0 (simplexml usage)
  3. Curl (preferred) or url_fopen or fsockopen should be available on your server.
  4. ftp and founder admin access on your phpbb installation.

Installation

New 1.1.2 install guide

  1. Unzip the bbdkp zip file into /store/mods/
  2. Launch automod, choose the install link. this will copy all the files, perform the necessary edits.
  3. Copy these folders to the images folder : this is not done by Automod because it can't handle it due to the size
    from="root/images/class_images/*.*" to="images/class_images/*.*"
    from="root/images/bossprogress/*.*" to="images/bossprogress/*.*"
    from="root/images/roster_portraits/*.*" to="images/roster_portraits/*.*"
    from="root/images/roster_classes/*.*" to="images/roster_classes/*.*"
  4. Then surf to /install, and you will see the umil installation page. This will install the acp modules, database tables, and clear the caches (template, theme, imagesets)

Upgrading from 1.1.1 to 1.1.2

  1. There are two file packages : a full MODX package and a changed files MODX package. both are Automod Compatible
  2. After having updated the files, surf to /install, and you will see the Umil updater. It will give you the option to Upgrade the Db from 1.1.1, updating the acp modules, database tables, and clear the caches (template, theme, imagesets)

Upgrading from 1.1.0 to 1.1.2

  1. Using Automod, do an uninstall of 1.1.0 This will undo all the changes to the php and html files, leaving the bbdkp database tables.
  2. Unzip the bbdkp 1.1.2 zip file into /store/mods/
  3. Launch automod, choose the install link. this will copy all the files, perform the necessary edits. .
  4. Copy these folders to the images folder : this is not done by Automod because it can't handle it due to the size
    from="root/images/class_images/*.*" to="images/class_images/*.*"
    from="root/images/bossprogress/*.*" to="images/bossprogress/*.*"
    from="root/images/roster_portraits/*.*" to="images/roster_portraits/*.*"
    from="root/images/roster_classes/*.*" to="images/roster_classes/*.*"
  5. Then surf to /install, and you will see the Umil installer. It will give you the option to Upgrade the Db from 1.1.0, updating the acp modules, database tables, and clear the caches (template, theme, imagesets)

Upgrading from 1.0.8 (any) or 1.0.9rc1 to 1.1.2

  1. Note : if you have 1.09 beta you need to update your database to 1.0.9rc1 before doing an upgrade to 1.1.2
  2. Using Automod, do an uninstall of 1.09rc1. This will undo all the changes to the php and html files, leaving the bbdkp database tables. be sure to check that your phpbb files are clean.
  3. Unzip the bbdkp 1.1.0 into /store/mods/
  4. Launch automod, choose the install link. this will copy all the files, perform the necessary edits.
  5. Copy these folders to the images folder : this is not done by Automod because it can't handle it due to the size
    from="root/images/class_images/*.*" to="images/class_images/*.*"
    from="root/images/bossprogress/*.*" to="images/bossprogress/*.*"
    from="root/images/roster_portraits/*.*" to="images/roster_portraits/*.*"
    from="root/images/roster_classes/*.*" to="images/roster_classes/*.*"
  6. Then surf to /install, and you will see the Umil installer. It will give you the option to Upgrade the Db from 1.0.9rc1, updating the acp modules, database tables, and clear the caches (template, theme, imagesets)

EQDKP conversion for Wow and EQ1/EQ2 (is not included, separate download)

You can import your existing EqDkp tables into bbDkp. for this,
  1. Clean-install the bbdkp Mod on a phpbb3 forum as described above.
    • for importing EqDkp 1.3.2, launch /install/import_eqdkp132.php
    • for importing EqDkp 1.4.0, launch /install/import_eqdkp40.php
    • for importing EqDkp Plus, launch /install/import_eqdkpplus.php

Quick Deactivation

  1. should you wish to Deactivate this Mod quickly you can open config.php
    2. change @define('EMED_BBDKP', true); to //@define('EMED_BBDKP', true);
